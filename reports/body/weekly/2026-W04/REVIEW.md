## 💡 重要な知見

### 炭水化物 vs タンパク質の優先順位

**結論: 炭水化物を優先すべき**

理由:
```
1. カロリー不足が最大の問題
   → タンパク質をいくら摂ってもエネルギー不足で筋合成されない
   → むしろエネルギー源として消費される（糖新生）

2. Protein Sparing Effect（タンパク質節約効果）
   → 炭水化物を十分摂取すると、タンパク質が本来の役割（筋合成）に使われる

3. トレーニングパフォーマンスへの影響
   → 炭水化物不足で筋グリコーゲン枯渇
   → トレーニング強度低下
   → ボリューム減少（W04: 19,033kg）
```

### 脂質の重要性

```
脂質 1g = 9kcal（炭水化物・タンパク質は4kcal/g）
→ 少量で大きくカロリーを稼げる
→ 増量に最適

追加メリット:
✓ テストステロン生成に必要（筋肥大に重要）
✓ 脂溶性ビタミン（A,D,E,K）の吸収促進
✓ 満腹感が高い
✓ バターは常温保存可能
```

## 📝 まとめ

### 問題の本質

トレーニング頻度よりも、**栄養摂取の不足**が最大の問題です。現状は「減量中」の状態であり、このままでは筋肉が増えることはありません。

### 優先順位

1. **栄養摂取の改善（最優先）**
   - 総カロリー: 2300-2700kcal/日
   - タンパク質: 100g以上/日
   - 炭水化物優先でカロリー確保

2. **トレーニング頻度の安定化**
   - 最低週3回、理想は週4〜5回
   - 週間ボリューム45,000kg以上

3. **回復の改善**
   - 睡眠7〜8時間
   - HRVベースでの強度調整

### 重要なポイント

```
✓ 炭水化物でカロリーを確保してから、タンパク質を摂る
✓ 脂質（バター）を活用して効率的にカロリーを増やす
✓ 液体カロリー（豆乳）で無理なく摂取量を増やす
✓ 既存の習慣（オートミール）に追加するだけ
✓ 食事記録の徹底
```
