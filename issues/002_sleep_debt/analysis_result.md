## 睡眠必要量の正しい推定

### 問題点：従来の計算方法

従来のプロトタイプは「**習慣的睡眠時間（6.3時間）**」を睡眠必要量として算出していました。

**これには重大な欠陥があります**：
- 過去90日間ずっと睡眠不足だった場合、その不足状態が「正常」として認識される
- 慢性的睡眠不足を見逃してしまう
- 真の睡眠必要量ではなく、「習慣的な睡眠時間」を測定している

### 改善版：複数手法による推定

| 手法 | 推定値 | 信頼度 | サンプル数 | 説明 |
|------|--------|--------|-----------|------|
| **習慣的睡眠時間** | 6.3h | ❌ Low | 38日 | 現在のパターン（推奨範囲未満） |
| **HRVベース** | 6.6h | Medium | 9日 | HRV上位25%の日の睡眠時間 |
| **睡眠効率ベース** | 6.5h | Medium | 15日 | 睡眠効率上位25%の日 |
| **一般推奨値** | 8.0h | High | - | 米国睡眠財団の推奨 |
| **統合推奨値** | **7.2h** | ✅ **High** | - | **複数手法の重み付き平均** |

#### 統合推奨値の算出方法

```
重み付き平均:
  一般推奨値（8.0h）        × 重み 3.0  ← 科学的根拠を重視
  HRVベース（6.6h）         × 重み 2.0  ← 回復良好日のパターン
  睡眠効率ベース（6.5h）     × 重み 1.5  ← 高効率日のパターン
  習慣的睡眠（6.3h）         × 重み 0.5  ← 推奨範囲外なので重み小

結果: 7.2時間
```

**なぜこの方法が適切か**:
1. 科学的根拠（一般推奨値）を最も重視
2. あなたの実データ（HRV、睡眠効率）も反映
3. 習慣的睡眠は参考程度（推奨範囲外のため重みを下げる）
4. 推奨範囲（7-9時間）内に収まる

## 科学的根拠

### 推奨睡眠時間

**米国睡眠財団（National Sleep Foundation）**:
- 成人（18-64歳）: 7-9時間
- 高齢者（65歳以上）: 7-8時間

### 慢性的睡眠不足の健康リスク

**7時間未満の睡眠を継続すると**:
- 心血管疾患のリスク: +45%
- 肥満のリスク: +55%
- 2型糖尿病のリスク: +37%
- 高血圧のリスク: +71%（睡眠変動性が高い場合）

### 睡眠負債の回復

**研究結果**:
- 1時間の睡眠負債 → 回復に3-4日必要
- 慢性的な睡眠制限（10日間） → 1週間では完全回復しない
- 週末の寝だめは限定的な効果

## 実装の技術的改善

### 問題のあったアルゴリズム（v1）

```python
# 過去90日間の中央値を「睡眠必要量」とする
sleep_need = median(過去90日間の睡眠時間)  # 6.3時間

# 問題点:
# もし過去90日間ずっと睡眠不足だったら、
# その不足状態が「正常」として認識される
```

### 改善版アルゴリズム（v2）

```python
# 複数手法で推定し、重み付き平均を取る
estimates = {
    '一般推奨値': (8.0時間, 重み 3.0),
    'HRVベース': (6.6時間, 重み 2.0),  # 回復良好日
    '睡眠効率ベース': (6.5時間, 重み 1.5),  # 高効率日
    '習慣的睡眠': (6.3時間, 重み 0.5),  # 推奨範囲外は重み小
}

sleep_need = weighted_average(estimates)  # 7.2時間

# 推奨範囲内にクリップ
sleep_need = clip(sleep_need, 7.0, 9.0)
```

### 慢性的睡眠不足の検出

```python
# 推奨範囲未満の日数を計算
below_min_pct = (睡眠時間 < 7時間).sum() / 総日数 * 100

if below_min_pct > 70:
    severity = '重度'
elif below_min_pct > 50:
    severity = '中程度'
elif below_min_pct > 30:
    severity = '軽度'
else:
    severity = 'なし'
```
