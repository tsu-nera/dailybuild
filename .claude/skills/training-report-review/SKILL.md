指定されたレポート（REPORT.md）を分析してください。

## 体組成レポートの場合

以下の観点で評価:

1. **体重・筋肉量・体脂肪率の変化トレンド**
   - 体重: 月間+0.75kg目標に対する進捗
   - 筋肉量: 増加傾向か（増量期の場合）
   - 体脂肪率: 15%以下を維持しているか

2. **FFMI進捗**
   - 目標FFMI 21.0に対する現在値
   - 到達までの予測期間

3. **カロリー収支のバランス**
   - 摂取カロリーと消費カロリーのバランス
   - TDEE（総消費エネルギー）の妥当性
   - タンパク質摂取量（体重×2g推奨）

4. **改善アドバイス**
   - 増量/減量/維持フェーズの判断
   - カロリー調整の提案
   - 栄養バランスの改善点

## ワークアウトレポートの場合

以下の観点で評価:

### 前提条件
- **トレーニング環境**: チョコザップ（マシン限定、種目数が制約）
- **利用可能な種目**:
  - プッシュ系: ショルダープレス、チェストプレス、ディップス
  - プル系: ラットプルダウン、バイセップスカール
  - 下半身: レッグプレス
- **制約**: フリーウェイト不可、新規種目追加困難

### 評価観点

1. **Training Volume（重量×回数）の週次推移**
   - 前週比の増加率（週5-10%が理想、10%超は要注意）
   - 停滞している種目の特定
   - **トレーニング頻度の安定性**（週5回を基準、3回以下は警告）

2. **種目別の進捗評価**
   - 各種目のボリューム増加傾向
   - 重量の伸び（プログレッシブオーバーロード）
   - **特にラットプルダウンの重量進捗**（背中トレ不足リスク）

3. **バランスチェック（制約考慮版）**
   - 上半身/下半身のバランス
   - 引く動作/押す動作のバランス
     - **理想**: 引く/押す = 1.5-2.0
     - **チョコザップ現実値**: 引く/押す = 0.6-0.8を目指す
     - **最低ライン**: 引く/押す = 0.5以上（0.5未満は要改善）
   - 筋肉グループの偏りチェック

4. **チョコザップ制約内での改善提案**
   - 既存6種目の**セット数・重量s調整**で最適化
   - 新規種目追加は提案しない
   - 利用可能な種目内での**ボリューム配分変更**を推奨

5. **ボリューム増加の安全性評価**
   - 急激な増加による怪我リスク
   - 適切な回復期間の確保
   - **週2回などの極端な低頻度週の警告**

6. **次週のプログラム提案**
   - ボリューム増加の推奨値
   - 弱点部位の強化（セット数・重量で調整）
   - デロード（休養）週の必要性判断
   - **トレーニング頻度の維持策**（カレンダー固定など）

## 分析基準

### 体組成
- **筋肥大目標**: 月間体重+0.75kg、筋肉量増加
- **体脂肪率**: 15%以下を維持（クリーンバルク）
- **FFMI目標**: 21.0（現在18.4-18.5から段階的に）

### トレーニング
- **頻度**: 週5回を基準（最低4回、3回以下は警告）
- **ボリューム増加**: 週5-10%が理想（10%超は過負荷リスク）
- **バランス比率（チョコザップ版）**:
  - 理想: 引く/押す = 1.5-2.0（通常ジム）
  - **現実目標**: 引く/押す = 0.6-0.8（チョコザップ）
  - **最低ライン**: 引く/押す = 0.5以上（姿勢・肩の健康）
- **プログレッシブオーバーロード**: 各種目で定期的な重量・回数増加

### 回復
- **HRV・睡眠データ**: レポートに含まれる場合は考慮
- **デロード**: 3-4週間ごとに計画的な軽減週

## 出力形式

1. **現状サマリー**: 主要指標の要約
2. **良い点**: 順調な進捗項目
3. **改善点**: 課題となる項目
4. **具体的アドバイス**: 次のアクション（数値目標含む）
