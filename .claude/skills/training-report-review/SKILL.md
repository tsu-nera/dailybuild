指定されたレポート（REPORT.md）を分析してください。

## 体組成レポートの場合

以下の観点で評価:

1. **体重・筋肉量・体脂肪率の変化トレンド**
   - 体重: 月間+0.75kg目標に対する進捗
   - 筋肉量: 増加傾向か（増量期の場合）
   - 体脂肪率: 15%以下を維持しているか

2. **FFMI進捗**
   - 目標FFMI 21.0に対する現在値
   - 到達までの予測期間

3. **カロリー収支のバランス**
   - 摂取カロリーと消費カロリーのバランス
   - TDEE（総消費エネルギー）の妥当性
   - タンパク質摂取量（体重×2g推奨）

4. **改善アドバイス**
   - 増量/減量/維持フェーズの判断
   - カロリー調整の提案
   - 栄養バランスの改善点

## ワークアウトレポートの場合

以下の観点で評価:

1. **Training Volume（重量×回数）の週次推移**
   - 前週比の増加率（週5-10%が理想、10%超は要注意）
   - 停滞している種目の特定

2. **種目別の進捗評価**
   - 各種目のボリューム増加傾向
   - 重量の伸び（プログレッシブオーバーロード）

3. **バランスチェック**
   - 上半身/下半身のバランス
   - 引く動作/押す動作のバランス（引く/押す = 1.5-2.0推奨）
   - 筋肉グループの偏りチェック

4. **ボリューム増加の安全性評価**
   - 急激な増加による怪我リスク
   - 適切な回復期間の確保

5. **次週のプログラム提案**
   - ボリューム増加の推奨値
   - 弱点部位の強化メニュー
   - デロード（休養）週の必要性判断

## 分析基準

- **筋肥大目標**: 月間体重+0.75kg、筋肉量増加
- **ボリューム増加**: 週5-10%が理想（10%超は過負荷リスク）
- **バランス比率**: 引く/押す = 1.5-2.0推奨（姿勢改善・肩の健康）
- **回復指標**: HRV・睡眠データも考慮（レポートに含まれる場合）

## 出力形式

1. **現状サマリー**: 主要指標の要約
2. **良い点**: 順調な進捗項目
3. **改善点**: 課題となる項目
4. **具体的アドバイス**: 次のアクション（数値目標含む）
